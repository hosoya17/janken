## 開発の概要
Windows上で動作するじゃんけんができるアプリです。<br>
## システムの概要
・playerの点数が0以下になるか、CPUの点数が0以下になると終了します。<br>
・playerが勝つとplayerに3点加点、CPUは2点減点されます。<br>
・CPUが勝つとCPUに3点加点、playerは2点減点されます。<br>
・あいこはどちらも点数に変動はありません。
